Hugo学习指南

Written by Bellamy with ♥ on July 14, 2019 in 博客学习

一、安装hugo

准备

  • 文件操作
    1. 创建一个新文件夹，D:\Hugo
    2. 创建一个新文件夹，D:\Hugo\bin
    3. 创建一个新文件夹，D:\Hugo\Sites
  • 下载hugo
    1. 在浏览器中打开 https://github.com/spf13/hugo/releases。
    2. 当前的版本是 hugo_0.55.6_Windows-64bit.zip。
    3. 下载那个 ZIP 文件，并保存到 D:\Hugo\bin 文件夹中。
    4. 解压之后得到三个文件hugo.exe、LICENSE、README.md
    5. 确保 hugo.exe 文件在 D:\Hugo\bin 文件夹。（如果你没有在D:\Hugo\bin 请把它移动去）
    6. 把 hugo.exe 可执行文件添加到你的 PATH路径中。把hugo添加到path中的操作：
      我的电脑邮件右键属性，在高级里面，选择环境变量，在系统变量里面找到path。双击打开path编辑，新建路径输入D:\hugo\bin（作者这里是D:\hugo\bin，其他的自己改路径）

二、部署hugo

  1. 验证：打开cmd，用hugo help 验证，有提示输出则为安装成功
  2. 输入cmd命令
    D:\Hugo\Sites> hugo new site 你的昵称.org
  3. 建网站
    1. 用cmd输入(bellamy0处改为你自己站点的名字):
      D:\Hugo\Sites> hugo new site bellamy0.org
      这样就会在sites目录里面生成bellamy0.org的文件夹用来存放网站文件等。
    2. 创建about页面，输入：
      hugo new post/first.md
    3. 创建第一篇文章，输入：
      hugo new about.md
    4. 打开 post/first.md 编辑blog内容。
      发布时需要将draft：true改为draft：false

  4. 安装皮肤

    1. 在站点目录下生成themes文件夹，随后用cmd在站点下运行：
      cd themes
      git clone + 皮肤的github地址

    皮肤下载地址：https://www.gohugo.org/theme/
    根据不同皮肤的提示安装。

三、部署到github

1.在GitHub上选择右上角 new repository，第一行待输入处输入:你的GitHub用户名.github.io，点击下方绿色生成。
2.点击最上方setting，下翻至大标题github pages，选择source中的master分支
3.将个人博客部署到远端，cmd在站点目录下输入命令
hugo --theme=你的主题名 --baseUrl="https://你的仓库名.github.io/" --buildDrafts
4.cmd中进入public文件夹，输入：
git init
git add .
git commit -m "firstcommit"
git remote add origin https://github.com/yourgithubname/仓库地址.git
git push -u origin master
按要求输入用户名和密码，即成功

四、问题

  • 中文乱码：修改编码方式为UTF-8

五、后续修改方式

  1. 用cmd进入站点，执行：
    hugo
  2. 进入public执行：
    git add -A
    git commit -m "xxxx"
    git push -u origin master
