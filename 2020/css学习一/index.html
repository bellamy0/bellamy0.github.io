CSS学习一

Written by Bellamy with ♥ on February 4, 2020 in 前端学习

一.CSS基础语法

  • 格式： 选择器{属性1：值1；属性2：值2}
  • 基本样式：width、height、background-color
  • 长度单位：px（像素）、%（相对于外容器的百分比）
  • css注释：/* 注释内容 */
  • 形式
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <meta http-equiv="X-UA-Compatible" content="ie=edge">
    <title>Document</title>
    <style>     //<style> 内写css，此处为设置下方的div
        div{ width:100px;height:100px;background-color:red }
    </style>
</head>
<body>
    
</body>
</html>

二.css样式的引入方式

  • 内联样式：style属性
  • 内部样式：style标签（可以复用标签，更优，看起来更简洁）
  • 形式
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <meta http-equiv="X-UA-Compatible" content="ie=edge">
    <title>Document</title>
    <style>     //<style> 内写css，此处为设置下方的div,内部样式
        div{ width:100px;height:100px;background-color:red }
    </style>
</head>
<body>
    //内联样式
    <div style="width:100px;height:100px;background-color:red">
        这是一个块
    </div>
</body>
</html>

  • 外部样式：引入css文件

  • 引入方式：

    • 法一：<link>引入外部资源

    • 属性：rel（指定资源与页面的关系）、href（资源的地址）

    • 法二：@import（不建议）

    <head>
    <meta charset="UTF-8">
    <link rel="stylesheet" href="./common.css"> //法一
    <title>Document</title>
    <style>
        @import url('./common.css');
    </style>
</head>
    • css文件形式
    div{ width:100px;height:100px;background-color:red }

三.css中的颜色表示法

  • 单词表示法：transparent
  • RGB表示法：rgb(0,0,0)~rgb(255,255,255)
  • 十六进制表示法：#000000~#ffffff
  • 提取颜色的下载地址： https://www.baidufe.com/fehelper

四. css背景样式

  • background-color: 背景颜色
  • background-image:背景图片
    • url（背景地址） 默认会水平都铺满
  • background-repeat:平铺方式
    • repeat-x x轴平铺
    • repeat-y y轴平铺
    • repeat x、y轴都平铺——默认值
    • no-repeat 都不平铺
  • background-position:背景位置x y
    • 用数字表示：px或%
    • 用单词表示：
      • x： left、center、right
      • y：top、center、bottom
  • background-attachment:背景图随滚动条移动的方式
    • scroll 按照当前元素进行偏移——默认值
    • fixed 按照浏览器进行偏移
<style>
    body{ height: 2000px;} 
    div{
        width:1440px;height:800px;background-color:red;
        background-image:url(./img/dog.jpg);
        background-repeat:no-repeat;
        background-position:50% 50%;
        background-attachment:fixed;
    }
</style>

//背景实现视觉差效果    
<head>
    <style>
        //用#+div+name 关联某个容器
        #div1{ width:1440px;height:800px; background-image:url(./img/1.jpg);}
        #div2{ width:1440px;height:800px; background-image:url(./img/2.jpg);}
        #div3{ width:1440px;height:800px; background-image:url(./img/3.jpg);}
    </style>
</head>
<body>
    <div id="1"></div>
    <div id="2"></div>
    <div id="3"></div>
</body>

五.css边框样式

  • border-style:边框样式
    • solid:实线
    • dashed:虚线
    • dotted:点线
  • border-width:边框大小
  • border-color:边框颜色
  • 针对某一个边的单独设置： border-right/left/top/bottom-style……

六. CSS文字样式

  • font-family:字体类型

    • 字体类型中存在空格时要加引号：’Times New Roman’、微软雅黑

    • 字体间用”，“设置多种字体，前一种无法显示时寻找后一种显示

  • font-size：字体大小——默认16px

    • 字体大小建议为偶数px，更好实现文字对齐

  • font-weight:字体粗细

    • 单词设置
    • normal：正常
    • bold：加粗
    • 数值设置：
    • 100、200、……、900
    • 实际显示效果100-500为正常、600-900为加粗

  • font-style: 字体样式

    • normal：正常
    • italic：斜体

  • font-color：字体颜色

七.文本

1.css段落样式

  • text-decoration: 文本修饰，通过空格添加多个文本修饰
    • underline: 下划线
    • line-through: 删除线
    • overline: 上划线
    • none：不添加任何
  • text-transform：设置大小写
    • lowercase：小写
    • uppercase：大写
    • capitalize：首字母大写
  • text-indent：文本缩进
    • number+px ：绝对像素
    • number+em: 相对于font-size的大小，首行缩进2个则为2em
  • text-align:文本对齐方式
    • left、right、center、justify（两端对齐）
  • line-height：定义行高number+px 或number（表示比例值）
    • 默认行高不是固定值
  • letter-spacing：字体间距number+px
  • word-spacing：词之间的间距
  • word-break：break-all；（非常强烈的折行）
  • word-wrap：break-word；（不强烈的折行，会产生空白）

八.css复合样式

  • 复合的写法，是通过空格的方式实现的，有的不用关心顺序：如background、border。有的需要按顺序写：如font
  • background：red url（） repeat 0 0；
  • border：1px red solid；
  • font：size famliy 至少有两个值 大小与字体类型（顺序不可调）
    • weight style size/line-height famliy前面可以任意 后俩位置固定
    • color不能加
  • 单一与复合混用必须：先复合再单一 ！
