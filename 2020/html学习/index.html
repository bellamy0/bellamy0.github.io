一.VSCode编辑器

快捷键的使用

ctrl + a:全选
ctrl + z、ctrl + y: 撤销、前进
shift + end：从头选中一行
shift + home：从尾部选中一行
shift + alt + ↓：快速复制一行
alt + ↑或↓：快速移动一行
单词+tab：快速形成标签<单词>
!+tab:快速创建html初始代码
ctrl+/:快速生成/删除注释
shift+alt+a：快速生成/删除注释

二.html初始代码

<!DOCTYPE html>      //文档声明：html文件
    <html lang="en">    //html文件最外层标签：初始为英文网站(lang="zh-CN"表示中文)
        <head>       //头部（通常为输入网址的那一栏）
            <meta charset="UTF-8">  //元信息：编写网页中的赋值信息，此处为采用国际编码
            <title>Document</title> //头部处的标题
        </head>
    <body>  
        //页面中的所有内容
    </body>
</html>`

三.标题与段落

1.标题

  • 双标签（<h1></h1>……<h6></h6>)

  • 一个页面中只能有一个h1，其余标题无限制。

  • h5、h6在开发中不常用

2.段落

  • 双标签（<p></p>）

3. 文本修饰标签

  • <strong></strong>:加粗
  • <em></em>:斜体
  • <sub></sub>:下标
  • <sup></sup>:上标
  • <del></del>:删除文本
  • <ins></ins>:插入文本

四.图片标签与图片属性

1.图片标志

  • <img>单标签

2.形式

  • <img src=“” alt=“”>

2. 图片属性：

  • src：引入图片地址
  • alt：当图片出现问题时，显示一段图片提示文字
  • title: 提示信息（当鼠标移到图片上时）
  • width、height ：设置大小 (尽量加上宽高，能给用户更好体验)

五.引入文件的地址路径

1.相对路径

  • 相对于某个文件

  • ”.“表示当前路径

  • ”..“表示上一级路径

2.绝对路径

  • 直接写整个地址、可去掉盘符用/开头

***注意尽量写”/“而非”\“

六.跳转链接

1. 标志

  • <a></a>

2.形式

<a herf="http://www.baidu.com">访问百度</a> //用文字设置链接
<a herf="http://www/baidu.com">
    <img src=".img/dog.jpg" alt=""> //用图片设置链接
</a>

3.属性

  • href属性：链接地址
  • target属性：可以改变链接打开的方式，默认为在当前页面打开：_self,在新页面打开为：_blank

4.改变链接默认行为的标签

  • <base>

  • 写在用于设置的<head>中,为单标签

  • 形式

<head>
    <base target="_blank">
</head>
<body>
</body>

七.跳转锚点

1.标志

  • <a></a>

2.实现方法

  • #号 + id属性（id写在<h>内）
  • #号 + name属性（name写在<a>内）

2.形式

<body>
    //法一
    <a href="#html">HTML</a>  //用href链接标题id，前面加#  
    <h2 id="html">HTML超文本标记语言</h2>
    <p>模拟的段落</p>
    
    //法二
    <a href="#html">HTML</a>  //用href链接标题id，前面加# 
    
    <a name="html"></a>
    <h2 id="html">HTML超文本标记语言</h2>
    <p>模拟的段落</p>
</body>

八. 列表标签

1.无序标签

  • 标志：<ul></ul>&<li></li>

  • <ul>与<li>中间不能有别的标签,但是内部可以有别的标签，如

<ul type="value">

  • 有嵌套规范

  • 形式

<body>
    <ul>
        <li>第一项</li>
        <li>第二项</li>
    </ul>
</body>

2.有序列表

  • 标志：<ol></ol>&<li></li>
  • 其余与无序列表相同
  • 可用type改变 排序方式（用数字or罗马数字or英文字母排序）

3.自定义列表

  • 标志：<dl><\dl>、<dt></dt>、<dd></dd>
  • 形式
<body>
    <dl>
        <dt>HTML</dt>
        <dd>超文本标记语言</dd>
    </dl>
</body>
  • 有标题与标题内容的列表

4. 嵌套列表

  • 形式
<body>
    <ul>
        <li>
            四川省
            <ul>
                <li>成都市</li>
                <li>绵阳市</li>
            </ul>
        </li>
    </ul>
</body>

九.表格

1.表格标签

  • <table>:表格最外层容器
  • <tr>:定义表格行
  • <th>:定义表头
  • <td>:定义表格单元
  • <caption>:定义表格标题

2.形式

<body>
    <table>
        <caption>天气预报</caption>
        <tr>
            <th>日期</th>
            <th>天气情况</th>
            <th>出行情况</th>
        </tr>
        <tr>
            <td>2020年2月3日</td>
            <td><img src="" alt=""></td>
            <td>天气晴朗</td>
        </tr>
    </table>
</body>

3.语义化标签

  • 更符合html规范 有利于合作：<tHead>、<tBody>、<tFood>
<body>
    <table>
        <caption>天气预报</caption>
        <tHead>
            <tr>
                <th>日期</th>
                <th>天气情况</th>
                <th>出行情况</th>
            </tr>
        </tHead>
        <tBody>
            <tr>
                <td>2020年2月3日</td>
                <td><img src="" alt=""></td>
                <td>天气晴朗</td>
            </tr>
        </tBody>
        <tFood>
            
        </tFood>
    </table>
</body>

注：在一个table中 ，tBody是可以多次出现的，tHead、tFood只能出现一次

4.表格属性

  • border：表格边框(加在<table>里)
  • cellpadding：单元格内的空间(加在<table>里)
  • cellspacing：单元格之间的空间(加在<table>里)
  • rowspan：合并行(加在<td>里)
  • colspan：合并列(加在<td>里)
  • align：左右对齐方式（right,left,center,加在<tr>里）
  • valign：上下对齐方式（top,middle,bottom,加在<tr>里）

十.表单

1.表单标签

  • <form></form>：表单的最外层容器
  • <input>：单标签，用于搜集用户信息，根据不同的type值展示不同的控件
type属性 含义
text 普通的文本输入框
password 密码输入框
checkbox 复选框
radio 单选框
file 上传文件
submit 提交按钮
reset 重置按钮
  • <textarea></textarea>：多行文本框
  • <select>……<option></option>……</select>:下拉菜单
  • <lable></lable>：辅助表单，提升用户体验（点字也能选中）

形式

<body>
    <form action="">    //action为提交按钮的提交地址
        <h2>输入框：</h2>
        <input type="text" placeholder="请输入用户名">    //框内提示消息
        <h2>密码框：</h2>
        <input type="password" placeholder="请输入密码"> //框内提示消息
        <h2>复选框：</h2>
        <input type="checkbox" checked>苹果       //checked让该项在开始就被选中
        <input type="checkbox" disabled>香蕉      //disabled禁止使用
        <input type="checkbox">葡萄
        <h2>单选框：</h2>   //通过name使其成为一组，只能选一个
        <input type="radio" name="gender">男
        <input type="radio" name="gender">女
        <h2>上传文件</h2>
        <input type="file">
        <h2>提交按钮与重置按钮</h2>
        <input type="submit">
        <input type="reset">    //重置整个表单
        <h2>多行文本框</h2>
        <textarea cols="" rows=""></textarea>
        <h2>下拉菜单</h2>
        <select multiple size="">   //multiple支持多选,size一次显示的个数
            <option selected disabled>请选择</option>  //selected为默认选中项,disabled为不能再次选择项目
            <option>北京</option>
            <option>上海</option> 
            <option>杭州</option>
        </select>
        <h2>单选框：</h2>   //通过name使其成为一组，只能选一个
        <input type="radio" name="gender" id="man"><lable for="man">男</lable>
        <input type="radio" name="gender" id="woman"><lable for="woman">女</lable>
        //for与id映射
    </form>
</body>

十一.特殊字符

  • 空格：插入&nbsp;

十二.div与span

  • div（块）：划分区域的容器
  • span（内联）：修饰文字
