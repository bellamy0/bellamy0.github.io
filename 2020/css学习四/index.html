CSS学习四

Written by Bellamy with ♥ on February 8, 2020 in 前端学习

一.float浮动

  • 文档流：文档流是文档中可显示对象在排列时所占用的位置
  • float特性：加浮动元素（提高当前内容的层次），会脱离文档流（脱离body），会延迟父容器靠左或靠右排列，如果之前已经有浮动元素，会挨着浮动的元素进行排列
  • float取值：left、right、none（默认）
  • float注意点 ：
    • 只会影响后面的元素 ①
    • 内容默认提升半层（图文混排效果）
    • 宽由内容决定（类似内联）（块是由父容器决定）
    • 换行排列
    • 针对内联也适用（不常用）
    • 不方便居中
<head>
    <style>
        //此时黄色在上，红色在下，按文档流中的块排列  
        body{ border:1px black solid;}
        #box1{ width:100px; height:100px; background:yellow; }
        #box2{ width:200px; height:200px; background:red; }
        //此时黄色会浮在红色上方，依靠着由红色撑起来的父容器，向左排列
        body{ border:1px black solid;}
        #box1{ width:100px; height:100px; background:yellow; float:left; }
        #box2{ width:200px; height:200px; background:red; }
        //此时两者都浮动，文档流（body）部分无内容，两者并排
        body{ border:1px black solid;}
        #box1{ width:100px; height:100px; background:yellow; float:left; }
        #box2{ width:200px; height:200px; background:red; float:left; }
        //① 黄色依然在红色上方，区别是蓝色此时在红色底部
        body{ border:1px black solid;}
        #box1{ width:100px; height:100px; background:yellow;}
        #box2{ width:200px; height:200px; background:red; float:left; }
        #box3{ width:300px; height:300px; background:blue; }
    </style>
</head>
<body>
    <div id="box1"></div>
    <div id="box2"></div>
    <div id="box3"></div>
</body>

  • 清除浮动

    • 上下排列：clear属性：表示清除浮动的，值可以为left，both，right （clear属性只会操作块标签，对内联标签不起作用）
    <head>
    <style>
        //此时黄色在上，红色在下，按文档流中的块排列  
        body{ border:1px black solid;}
        #box1{ width:100px; height:100px; background:yellow; float:left; }
        #box2{ width:200px; height:200px; background:red; clear:left; }
    </style>
</head>
<body>
    <div id="box1"></div>
    <div id="box2"></div>
</body>

    • 嵌套排列：

    • 固定宽高：不推荐，不能把高度固定死，不适合自适应

    • 父元素设置浮动：不推荐，在同一个层次上虽然能撑开，但是会影响后面的内容

    • overflow:hidden：不推荐，内容溢出时会被截掉

    • display:inline-block： 不推荐，更改了盒模型的排列

    • 设置空标签：不推荐，用空标签清除浮动，会多添加一个标签

      <head>
      <style>
          #box1{ width:200px; border:1px black solid;}
          #box2{ width:100px; height:200px; background:red; float:left; }
          .clear{clear:both;}
      </style>
  </head>
  <body>
      <div id="box1">
        <div id="box2"></div>
          <div class="clear"></div>
      </div>
      aaaaaaaa
  </body>

    • 用after伪类：推荐，空标签加强版，（clear属性只会操作块标签，对内联标签不起作用）

      <head>
    <style>
        #box1{ width:200px; border:1px black solid;}
        #box2{ width:100px; height:200px; background:red; float:left; }
        .clear:after{ content:''; clear:both; display:block; }
    </style>
</head>
<body>
    <div id="box1" class="clear">
        <div id="box2"></div>
    </div>
    aaaaaaaa
</body>

  • 制造浮动又清除浮动的原因：制造浮动是为了让块元素并列显示，浮动容易打乱排版，清除浮动后仍然能并列显示

二.relative相对定位等

  • position定位：

    • 取值：static（默认），relative，absolute，fixed，sticky

    • relative相对定位（不可层叠）：(上下左右是反的)

    • 如果没有定义偏移量，对元素没有任何影响

    • 不使元素脱离文档流（会占位）

    • 不影响其他元素布局

    • 是相对于当前元素自身位置偏移（左上角为（0，0））

    #box2{ width:100px; height:100px; background:red; position:relative; left:100px; }
//可设置left、right、top、bottom

    • absolute绝对定位（可层叠）：

    • 使元素脱离文档流（类似浮动）

    • 可以让内联元素具备宽高（让内联具有块特性）

    • 使块元素默认宽高由内容决定（让块具有内联性）

    • 有定位祖先元素，则相对于祖先元素偏移，没有则相对于整个文档偏移

    • fixed固定定位：（返回顶部功能，弹窗广告功能）

    • 使元素脱离文档流（类似浮动）

    • 可以让内联元素具备宽高（让内联具有块特性）

    • 使块元素默认宽高由内容决定（让块具有内联性）

    • 相对于整个浏览器窗口进行偏移

    • sticky黏性定位：吸附效果（拖动到设置位置时会固定）

    • z-index：提升层级（取值整数），几个position属性都适用，同一层级优先比较

    z-index:0;

    三.css添加省略号

    • text-overflow:ellipsis; ：添加省略号
    • white-space:nowrap; ：内容不折行

    四.css精灵（雪碧图）

    • 特性：将网页涉及到的零星图片都包含到一张图上
    • 好处：减少图片的质量，网页的图片加载速度快，减少图片的请求次数

    五.css圆角设置

    • border-radius:number or %;：圆的半径
    <head>
<style>
    //四个角
    #box{ width:200px; height:200px; background:red; border-radius:100px; }
    //容器的百分比
    #box{ width:200px; height:200px; background:red; border-radius:50%; }
    //两对角
    #box{ width:200px; height:200px; background:red; border-radius:10px 20px; }
    //四个角
    #box{ width:200px; height:200px; background:red; border-radius:10px 20px 30px 40px; }
    //椭圆角
    #box{ width:200px; height:200px; background:red; border-radius:20px/40px; }
</style>
</head>
