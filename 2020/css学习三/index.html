CSS学习三

Written by Bellamy with ♥ on February 6, 2020 in 前端学习

一.CSS样式的继承

  • 文字相关样式能被继承
  • 布局相关样式不能被继承（默认不能，但是可以设置继承属性inherit）
<head>
    <style>
        //div标签的属性能被用于子标签p上——继承
        div{ width:300px; height:300px; color:red; border:1px; }
        //布局继承
        p{ border:inherit; }
    </style>
</head>
<body>
    <div>
        <p>这是一个段落</p>
    </div>
</body>

二.CSS的优先级

  • 相同样式：后面的优先级更高（不建议）
  • 内部样式与外部样式： 后引入的样式优先级高
  • 单一样式优先级：style行间>id>class>tag>*>继承
  • !important ：提升样式优先级（不建议使用）
  • 标签+类与单类：标签+类>单类⑤
  • 同一样式下群组与单一：不分优先级，按顺序，靠后的优先级高
  • 层次优先级：
    • 权重比较：累加数值之比
    • 约分比较
<head>
    <style>
        #elem{ color:red; }
        .box{ color: blue; }
        div{ color:blue; }
        *{ color:red; }
        //important的使用
        #elem{ color:red; !important; }
        //⑤
        div.box{ color:blue; }
        .box{ color:red; }
        //⑥
        div.p{ color:red; }
        div{ color:blue; }
    </style>
</head>
<body>
    <div id="elem" style="color:blue;">  //此处style优先级>id
        这是一个块
    </div>
    <div id="elem" class="box">  //此处id优先级>class
        这是一个块
    </div>
    <div id="elem" class="box">  //此处id优先级>class
        这是一个块
    </div>
</body>

三.CSS盒子模型

1.盒子模型基础

  • 组成：content（物品） -> padding（填充物） ->border（包装盒） ->margin（盒子与盒子之间的间距）

  • content：内容区域 width和height组成的

  • padding：内边距（内填充）,也可以分别设置padding-left/right/top/bottom

    • 只写一个值：30px（上下左右）

    • 写两个值：30px 40px（上下），（左右）

    • 写三个值（不建议）

    • 写四个值：30px 40px 50px 60px （上、右、下、左）

  • margin：外边距（外填充）与padding相同

  • 注意：

    • 背景色填充到margin以内的区域（不包括margin）
    • 文字会在content区域
    • padding不能为负数，而margin可以为负数

  • box-sizing: 盒尺寸，可以改变盒子模型的展示形态①

    • content-box（默认值）：width、height ——>作用于content
    • border-box：width、height——>作用于content、padding、border
    • 使用场景：
<head>
    <style>
        #box{
            width:200px; height:200px; background:red; border:10px blue solid;
            padding:30px 50px;
            box-sizing:border-box;//①
        }
    </style>
</head>
<body>
    <div id="box">这是一个盒子</div>
</body>

2.盒子模型的问题

2.1 盒模型的叠加问题：
  • 解决办法：
    • BFC规范
    • 直接改变间距
<head>
    <style>
        #box1{width:200px; height:200px; background:red; margin-bottom:30px}
        #box2{width:200px; height:200px; background:blue; margin-top:30px}
        //最终间距不是60px而是30px
        //叠加时会选取较大的一个作为margin值
    </style>
</head>
<body>
    <div id="box1">这是一个盒子</div>
    <div id="box2">这是一个盒子</div>
</body>
2.2 margin传递问题
  • 出现在嵌套的结构中，只是针对margin-top的问题
  • 解决办法
    • BFC规范
    • 加边框border
    • 不用margin用padding（通过计算加给父元素）
<head>
    <style>
        #box1{width:200px; height:200px; background:red;}
        #box2{width:200px; height:200px; background:blue; margin-top:100px;} 
        
    </style>
</head>
<body>
    <div id="box1">
        <div id="box2"></div>
    </div>
</body>

3.盒子模型扩展

  • 自适应：margin:auto; 这个方式只能使左右可以自适应
  • 宽高不设置时对盒子的影响：会自动计算容器大小，节省代码（与父容器相同）

四.标签分类

1.按类型

  • block：块
    • div、p、ul、ol、p、h1……
    • 特性一：独占一行
    • 特性二：支持所有样式
    • 特性三：不写宽的时候，跟父元素的宽相同
    • 特性四：所占区域是一个矩形

  • inline：内联

    • span、a、em、strong、img……
    • 特性一：挨在一起的
    • 特性二：有些样式不支持，例如：width、height、margin、padding
    • 特性三：不写宽的时候，宽度跟内容有关
    • 特性四：所占区域不一定为矩形
    • 特性五：内联标签之间会有空隙，而空隙是由换行引起的

  • inline-block：内联块

    • input、select……

五.display显示框类型

  • display属性可以强行将标签换为block、inline、inline-block、none（不显示某项）
  • display:none; 与visibility:hidden;区别：
    • 前者不占空间的隐藏,后者为占空间的隐藏
<head>
    <style>
        div{ width:100px; height:100px; background:red; display:inline; }
        span{ width:100px; height:100px; background:red; display:block; }
    </style>
</head>
<body>
    <div>块1</div>
    <div>块2</div>
    <span>内联1</span>
    <span>内联1</span>
</body>

六.标签嵌套规范

  • 块能嵌套内联
  • 块不一定能嵌套块
  • 内联不能嵌套块标签（但有特殊a标签）

七.overflow溢出隐藏

  • overflow:visible：默认
  • overflow:hidden：溢出文字会隐藏 ，直接剪裁
  • overflow:scroll：添加滚动条

  • overflow:auto：根据内容多少自动判断是否添加滚动条

  • overflow-x overflow-y

八.透明度与手势

  • opacity:opacity:0~1 透明~不透明
    • 占空间
    • 子内容也会透明
  • rgba：background:rgba(num,num,num,0~1)
    • 可以让指定内容透明
  • cursor：手势（鼠标移入时的样式）
    • 默认箭头
    • cursor:pointer
    • 自定义手势：
    • 准备图片：.cur 、.ico
    • cursor:url(./img/cursor.ico),auto

九.最大最小宽高

  • min-width/height: 内容不能充满时最小容器宽高，超出时自适应

  • max-width/height:内容最大容器宽高，小于时自适应

  • %是根据父容器来判断的

  • 一个容器适应屏幕的高的方式：

    • html,body{ height:100%; }

    .contrainer{ height:100%; }

十.css的简单reset

  • *{ margin:0; padding:0; } :使用通配改变（低性能）

  • ul{ list-style:none; }：去掉列表的点

  • a{ text-decoration:none; color:#999; } :去掉下划线，更改颜色

  • img{ display:block; }：图片跟容器底部有空隙（内联元素的对齐方式是按基线对齐而非文字底线）

    • 法一：

    • img{ vertical-align:baseline; }:基线对齐，默认值

    • img{ vertical-align:bottom; }:底线对齐

    • 法二：

    • img{ display:block; }
