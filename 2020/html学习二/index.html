一.CSS基础语法

  • 格式： 选择器{属性1：值1；属性2：值2}
  • 基本样式：width、height、background-color
  • 长度单位：px（像素）、%（相对于外容器的百分比）
  • css注释：/* 注释内容 */
  • 形式
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <meta http-equiv="X-UA-Compatible" content="ie=edge">
    <title>Document</title>
    <style>     //<style> 内写css，此处为设置下方的div
        div{ width:100px;height:100px;background-color:red }
    </style>
</head>
<body>
    
</body>
</html>

二.css样式的引入方式

  • 内联样式：style属性
  • 内部样式：style标签（可以复用标签，更优，看起来更简洁）
  • 形式
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <meta http-equiv="X-UA-Compatible" content="ie=edge">
    <title>Document</title>
    <style>     //<style> 内写css，此处为设置下方的div,内部样式
        div{ width:100px;height:100px;background-color:red }
    </style>
</head>
<body>
    //内联样式
    <div style="width:100px;height:100px;background-color:red">
        这是一个块
    </div>
</body>
</html>

  • 外部样式：引入css文件

  • 引入方式：

    • 法一：<link>引入外部资源

    • 属性：rel（指定资源与页面的关系）、href（资源的地址）

    • 法二：@import（不建议）

    <head>
    <meta charset="UTF-8">
    <link rel="stylesheet" href="./common.css"> //法一
    <title>Document</title>
    <style>
        @import url('./common.css');
    </style>
</head>
    • css文件形式
    div{ width:100px;height:100px;background-color:red }

三.css中的颜色表示法

  • 单词表示法
  • RGB表示法：rgb(0,0,0)~rgb(255,255,255)
  • 十六进制表示法：#000000~#ffffff
