javascript学习三——DOM

Written by Bellamy with ♥ on February 21, 2020 in 前端学习

1.节点

  • DOM——>document

  • 节点

    • 文本节点：无标签包裹的
    • 元素节点：标签中的

  • 获取子节点：

    • childNotes：获取某标签下所有节点
    • children：获取某标签下元素节点
    • oUl.children
    • 只算第一层

  • 父节点：

    • 查找父节点：parentNode

  • 父节点应用：隐藏列表项

  <head>
      <script>
          window.onload=function(){
              var aA=document.getElementsByTypeName("a");
              
              for(var i=0;i<aA.lenght;i++){
                  aA[i].onclick=function(){
                      this.parentNode.display=hidden;
                  };
            };
          };
      </script>
  </head>
  <body>
      <ul>
          <li><a href="javascript:;">a</a></li>
          <li><a href="javascript:;">b</a></li>
          <li><a href="javascript:;">c</a></li>
      </ul>
  </body>
  • offsetParent：获取元素在页面上的实际位置
  • firstChild/lastChild：获取收尾子节点
  • nextSibling/previousSibling：兄弟节点

2.操作元素的方式：

  • 用点操作
  • 用方括号操作
  • DOM：（一般不用）
    • setAttribute(“属性名”,“值”)：设置某属性的值
    • getAttribute（”属性名”）：获取某属性的值
    • removeAttribute(“属性名”)：删除某属性的值
<head>
    <script>
        window.onload=function(){
            var oTxt=document.getElementById("txt1");
            var oBtn=document.getElementById("btn1");
            
            oBtn.onclick=function(){
                //法一：用点操作元素
                oTxt.value="adf";
                //法二：用方括号操作元素
                oTxt["value"]="adf";
                //法三：DOM
                oTxt.setAttribute("value","adf");
            };
        };
    </script>
</head>
<body>
    <input id="txt1" type="text">
    <input id="btn1" type="button" value="按钮">
</body>

3.用className选择元素

  • 封装getByClass函数
  • o开头表示：object
  • a开头表示：array
  • s开头表示：string
//oParent表示父标签，sclass表示所需要找的class
function getByClass(oParent,sClass){
    //*为通配符，表示任何标签
    var aEle=oParent.getElementsByTagName("*");
    //用于放找到的元素
    var aResult=[];
    
    for(var i=0;i<aEle.length;i++){
        if(aEle[i].class==sClass){
            aResult.push(aEle[i]);
        };
    };
    return aResult；
};
//调用方式
var aBox=getByClass(oUl,"box");

4.DOM操作应用

4.1 创建DOM元素

  • var oLi = document.createElement("标签名");：仅为创建元素存入内存

  • oUl.appendChild(oLi);:为oUl元素添加子节点oLi到末尾（父级.appendchild(子节点)）

  • 实例：实现用输入框输入的文字生成新的表单项

<head>
    <script>
       window.onload=function(){
            var oBtn=document.getElementById("btn1");
            var oTxt=document.getElementById("txt1");
            var oUl=document.getElementById("ul1");
        
            oBtn.onclick=function(){
                var oLi=document.createElement("li");
                //将txt1中的数值放入表单项的内容中
                oLi.innerHTML=oTxt.value;
                oUl.appendChild(oLi);
            };
       };
    </script>
</head>
<body>
    <input type="text" id="txt1">
    <input type="button" id="btn1">
    <ul id="ul1"></ul>
</body>

4.2 插入DOM元素

  • oUl.insertBefore(oLi,在谁之前)：插入到某一个位置

4.3 删除DOME元素

  • oUl.removeChild(某元素)

4.4 文档碎片（用处极小）

  • 文档碎片可以提高低级浏览器上的DOM操作性能（先放入文档碎片中，最后统一进行渲染，而不用插一次元素渲染一次）

  • 文档碎片可以提高低级浏览器上的DOM操作性能（先放入文档碎片中，最后统一进行渲染，而不用插一次元素渲染一次）

  • var oFrag=document.createDocumentFragment();：创建文档碎片的语句

5.2 表格的添加与删除

                var oTr=document.creatElement("tr");

                oTd.innerHTML=id++;
                oTr.appendChild(oTd);
                
                var oTd=document.creatElement("td");
