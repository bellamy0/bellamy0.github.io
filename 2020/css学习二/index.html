一.ID选择器

  • css：#elem{}
  • html：id="elem"
<head>
    <style>
        #div1{ background: red;}
        #div2{ background: blue;}
    </style>
</head>
<body>
    <div id="div1">
        这是一个块
    </div>
    <div id="div2">
        这是另一个块
    </div>
</body>

  • 注意：

    • 在一个页面中ID值是唯一的，不可复用
    • 命名的规范：由字母、下划线、中划线组成（第一位不能是数字）
    • 快捷创建 容器名+#+id+tab
    • 小驼峰写法&短线写法&下划线写法

二.CLASS选择器

  • css：.elem{}
  • html：class="elem"
<head>
    <style>
        .box{ background: red;}
        .content{ font-size:30px;}
        //④.启用下面时会过滤掉div中的仅使p生效
        //p.box{ background:blue; }
    </style>
</head>
<body>
    <div class="box">
        这是一个块
    </div>
    //①.可复用
    <div class="box">
        这又是一个块
    </div>
    //②.可以添加多个class样式 
    <div class="box content">
        这又是一个块
    </div>
</body>

  • 注意：

    • 可以复用①
    • 可以添加多个class样式 （用空格）②
    • 添加多个样式时，优先级由css决定而非class属性中的顺序③
    • 标签+类的写法④
