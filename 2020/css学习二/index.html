CSS学习二

Written by Bellamy with ♥ on February 5, 2020 in 前端学习

一.ID选择器

  • css：#elem{}
  • html：id="elem"
<head>
    <style>
        #div1{ background: red;}
        #div2{ background: blue;}
    </style>
</head>
<body>
    <div id="div1">
        这是一个块
    </div>
    <div id="div2">
        这是另一个块
    </div>
</body>

  • 注意：

    • 在一个页面中ID值是唯一的，不可复用
    • 命名的规范：由字母、下划线、中划线组成（第一位不能是数字）
    • 快捷创建 容器名+#+id+tab
    • 小驼峰写法&短线写法&下划线写法

二.CLASS选择器

  • css：.elem{}
  • html：class="elem"
<head>
    <style>
        .box{ background: red;}
        .content{ font-size:30px;}
        //④.启用下面时会过滤掉div中的仅使p生效
        //p.box{ background:blue; }
    </style>
</head>
<body>
    <div class="box">
        这是一个块
    </div>
    //①.可复用
    <div class="box">
        这又是一个块
    </div>
    //②.可以添加多个class样式 
    <div class="box content">
        这又是一个块
    </div>
</body>

  • 注意：

    • 可以复用①
    • 可以添加多个class样式 （用空格）②
    • 添加多个样式时，优先级由css决定而非class属性中的顺序③
    • 标签+类的写法④

三.标签群组通配等选择器

1.标签选择器(TAG选择器)

  • css：标签{} eg：div{}
  • html：标签 eg：<div>
  • 使用场景：
    • 去掉某些标签的默认样式中
    • 复杂的选择器中，如 层次选择器

2.群组选择器(分组选择器)

  • css:用“，”隔开，如：div,#text,.title{ background-color=red; }

  • 通过逗号的方式实现代码复用

3.通配选择器

  • css：*{ }-> div,ul,li,p,h1,h2....{}

  • 注意：尽量避免使用通配选择器，会给所有标签添加样式

  • 使用场景：去掉所有or大部分标签的默认样式中

四.层次选择器

  • 后代选择器：所有后代都能被选择到①

M N {}

  • 父子：只有孩子能被选择到②

M > N {}

  • 兄弟：当前M下的所有兄弟N标签③

M ~ N {}

  • 相邻：当前M下面相邻的N标签④

M + N {}

实例：

<head>
    <style>
        //①
        #list li { background:red; }
        //②
        #list>li { background:red; }
        //③
        div ~ h2{ background:red; }
        //④
        div + h2{ background:red; }        
    </style>
</head>
<body>
    //①②
    <ul id="list">
        <li>
            <ul>
                <li></li>
            </ul>
        </li>
        <li></li>
        <li></li>
    </ul>
    <ol>
        <li></li>
        <li></li>
        <li></li>
    </ol>
    //③
    <h2>这是一个标题</h2>
    <div>这是一个块</div>
    <p>这是一个段落</p>
    <h2>这是一个标题</h2>
    <h2>这是一个标题</h2>
    //④
    <div>这是一个块</div>
    <h2>这是一个标题</h2>
</body>

五.属性选择器

实例

<head>
    <style>
        div[class]{ background:red; }
        //完全匹配
        div[class=box]{ background:red; }
        //部分匹配（部分匹配的都可以被选择）
        div[class*=search]{ background:red; }
        //起始匹配
        div[class^=search]{ background:red; }
        //结束匹配
        div[class$=search]{ background:red; }
        //多个中括号组合匹配
        div[class][id]{ background:red; }
    </style>
</head>
<body>
    <div>aaaa</div>
    <div class="box">bbbb</div>
    <div class="search">cccc</div>
    <div class="search-button">dddd</div>
    <div class="button-search">eeee</div>
    <div class="box" id="elem">ffff</div>
</body>

六.伪类选择器

  • css：M:伪类{}
  • 伪类用于向某些元素添加特殊效果
  • :link：访问前的样式 （只能给a标签）
  • :visited：访问后的样式（只能给a标签）
  • :hover：鼠标移入时的样式（可以给所有标签）
  • :active ：鼠标按下时的样式（可以给所有标签）
<head>
    <style>
        div{ width:200px;height:200px;background:red; }
        div:hover{ background:blue; }
        div:active{ background:green; }
        a:link{ color:green; }
        a:visited{ color:yellow; }
    </style>
</head>
<body>
    <div></div>
    <a href="#">这是一个链接</a>
</body>
  • 注意：
    • 四个伪类的顺序：L V H A
    • 一般网站只设置：a{} a:hover{}

七.after等伪类选择器

  • :after :befor :通过伪类的方式给元素添加一些文本内容（使用content属性分别为添加在前面与添加在后面）
  • :checked :disabled :focus :都是针对表单元素的
<head>
    <style>
        div:after{ content:world; color:red; }
        div:before{ content:world; }
        :checked{ width:100; height:100px; }    //被checked的会有该样式
        :disabled{ width:100; height:100px; }   //disabled会有该样式
        :focus{ background:red; }   //光标获取时有该样式
    </style>
</head>
<body>
    <div> hello!</div>
    <input type="checkbox">
    <input type="checkbox">
    <input type="checkbox" disabled>
    <input type="text"> 
</body>

八.结构性伪类选择器

  • :nth-of-type()：角标是从1开始的|n值表示从0-无穷大（nth-child 同理）
    • 区别：type：同一个父亲且样式中的第n个
    • 区别：child：同一个父亲下的第n个
  • :first-of-type()：第一个
  • :last-of-type()：最后一个
  • :only-of-type()：唯一一个
<head>
    <style>
        //使用方式
        li:nth-of-type(3){ background:red; }
        //实现隔行换色
        li:nth-of-type(2n){ background:red; }
    </style>
</head>
<body>
    <ul>
        <li></li>
        <li></li>
        <li></li>
        <li></li>
    </ul>
</body>
