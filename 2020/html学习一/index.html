一.VSCode编辑器

快捷键的使用

ctrl + a:全选
ctrl + z、ctrl + y: 撤销、前进
shift + end：从头选中一行
shift + home：从尾部选中一行
shift + alt + ↓：快速复制一行
alt + ↑或↓：快速移动一行
单词+tab：快速形成标签<单词>
!+tab:快速创建html初始代码
ctrl+/:快速生成/删除注释
shift+alt+a：快速生成/删除注释

二.html初始代码

<!DOCTYPE html>      //文档声明：html文件
    <html lang="en">    //html文件最外层标签：初始为英文网站(lang="zh-CN"表示中文)
        <head>       //头部（通常为输入网址的那一栏）
            <meta charset="UTF-8">  //元信息：编写网页中的赋值信息，此处为采用国际编码
            <title>Document</title> //头部处的标题
        </head>
    <body>  
        //页面中的所有内容
    </body>
</html>`
