javascript学习一——基础语法

Written by Bellamy with ♥ on February 16, 2020 in 前端学习

一、JS基础语法知识

1. JavaScript组成：

  • ECMAscript

  • DOM：文档

  • BOM：浏览器

2. 属性：

  • type=‘text/javascript’说明标签内的文本格式
  • src引入外部js文件

<script src="demo.js"></script>

3. 注意点：

  • 可以有多个<script>标签，但js是按代码顺序执行的
  • 一个标签只能做一件事情（要么引入，要么直接在里面写代码）
  • js代码需要有分号！

4. js中的输出方法：

  • alert()：弹出一个警告框
  • document.write()：在页面上输出内容
    • 如果括号内有html标签会自动解析，不会以文本形式输出
  • console.log()：在控制台上输入内容
    • 一般用于代码调试，不影响用户（招聘简介放在）

5、常量

  • 数据类型：
    • 数字：
    • 类型：number
    • 例子：100、-20、3.14
    • 布尔值：
    • 类型：Boolean
    • 例子：true、false
    • 字符串：
    • 类型：所有单引号或双引号的内容（都一样）
    • 例子：“100”，“hello”
    • 复合/引用数据类型：
    • 特殊数据类型：null、undefined（没有初始化）、NaN

6、变量

  • 声明变量关键字：var

7、变量弱引用

  • 定义：变量赋值是什么类型，就是什么类型
  • 注意：不建议改变变量的数据类型

8、三目运算符

num % 2 = 0 ? alert(偶数):alert(奇数)；

9、函数

  • 语法：声明函数
function 函数名(形参){}
  • 执行时需要用函数名(参数);调用
  1. 事件驱动函数

1.1 onclick：失去光标时触发

1.2 onblur：失去焦点时触发

  1. 匿名函数：无函数名，通过变量调用或直接调用
   <script>
       window.onload()=function(){
           //1.通过变量调用
           var obj1=function(){
               alert("通过变量调用");
           }
           obj1();
           
           //2.直接调用，函数必须加括号包裹，且让其执行需在尾部添加括号
           (function(){
               alert("直接调用")
           })();
       }
   </script>

10、字符串

  • 字符串重点方法：

    • substring：

    • 格式：字符串.substring(start,end);

    • 功能：将字符串中的[start,end]提出出来生成新的字符串

    • 返回值：新生成的字符串

    • substr：

    • 格式：字符串.substr(start,end)

    • 返回值：新生成的字符串

    • slice（是数组）

    • 格式：字符串.slice(start,end)

    • replace():

    • 格式：字符串.replace(oldstr,newstr)

    • 功能：用newstr替换oldstr

    • 参数：

      • 第一个参数传入的是字符串则只能替换一次

      • 第一个参数传入正则表达式/xxx/ig ，参数i代表忽略大小写，g代表全局匹配

      字符串.replace(/xxx/ig,newstr)

    • split()：

    • 格式：字符串.split(分隔符, length)

    • 参数：

      • 第一个参数，用这个分隔符对愿字符串进行分割
      • 第二个参数，控制返回数组的元素格式，一般不用

    • 注意：分隔符是空字符串时""，直接将每一个字符单独分割为字串返回

    • toLowerCase()

    • toUperCase()

    • concat:字符串拼接（一般情况下用+比较多）

    • 格式：字符串.concat(字符串1，字符串2，……)

    • indexof

  • 表单验证实例

二、js与html部分交互的部分函数

1.getElementId

  • 语法：x=document.getElementId("id")
  • 作用：用id返回（获取）某个标签

2.getElementsByTagName

  • 语法：var x=document.getElementByTagName("标签名")
  • 作用：返回所有该标签，每一个存入一个数组中，可以计算标签个数x.length

3.innerHTML

  • 语法：x.innerHTML
  • 作用：用于表示标签中的内容

4.onmouseover

  • 语法：可写入html作为一个属性、也可写入js中

<element onmouseover="">

object.onmouseover=function(){};

  • 作用：事件会在鼠标移入到指定元素上发生

5.onmouseout

  • 语法同上
  • 作用：事件会在鼠标移出到指定元素上发生

三、认识BOM

1.系统对话框

  • window方法（一般情况下window可以省略）
    • window.alert()：弹出警告框
    • window.confirm()：弹出一个带确定和取消的提示框（确定返回值为true，点击取消返回false）
    • window.prompt()：弹出一个带输入框的提示框
    • 参数：第一个参数：面板上显示的内容；第二个参数输入框里的默认（可以不填）
    • 返回值：点击确定返回输入框的内容，点击取消返回null
    • window.onload()=fuction(){}：在方法用于在网页加载完毕后立刻执行操作
  • open方法
    • 第一个参数：url，跳转该url
    • 第二个参数：字符串，给打开的窗口起一个名字，随后继续点则打开该名字的窗口
    • 第三个参数：一串特殊含义的字符串，可以设置很多属性……
